henk52-repo
===========

Repo conf administration

This module will disable the default fedora repositories and add the three mirror repositories.
 - Base
 - Upgrade
 - Everything

This is the repo module.


Installation:
sudo puppet apply /etc/puppet/modules/repo/tests/init.pp
